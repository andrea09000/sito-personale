# Portfolio personale — Andrea Savoldelli

Sito portfolio costruito con [Next.js](https://nextjs.org): one-page con progetti, esperienza, contatti e documentazione per la libreria [Savo.h](public/downloads/savo.h).

**Sito live:** [savoldelliandrea.me](https://savoldelliandrea.me)

## Avvio in locale

```bash
npm install
npm run dev
```

Apri [http://localhost:3000](http://localhost:3000).

## Build di produzione

```bash
npm run build
npm start
```

## Repository

Codice sorgente del portfolio Next.js; la libreria C `savo.h` è inclusa in `public/downloads/` e documentata in `/savo`.
